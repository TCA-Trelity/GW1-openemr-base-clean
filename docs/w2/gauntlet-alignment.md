# Gauntlet course-technique alignment map

*One row per technique taught in the course, the exact place it lives in this
codebase, and its status as of 2026-07-15. Every **shipped** row points at a
file that was opened and verified on that date — pointers, not claims. Other
statuses: **landing (CTn/Hn)** = in-flight this work cycle under the named
internal ticket id; **backlog (CTn)** = specified but deliberately unscheduled;
**deliberately not adopted** = a considered no, with the reason. This is an
index, not an argument — each row is checkable in one `git grep` or file open.*
